# BSS Group - Business Consulting Website

A professional, modern business consulting website built with HTML, CSS, and JavaScript.

## Features

### 🎨 Design
- **Modern & Responsive**: Fully responsive design that works on all devices
- **Professional Color Scheme**: Navy blue, sky blue, and gold accent colors
- **Smooth Animations**: CSS animations and scroll effects for engaging UX
- **Mobile-First Approach**: Optimized for mobile, tablet, and desktop

### 📑 Sections

1. **Navigation Bar**: Sticky header with logo and navigation links
2. **Hero Section**: Eye-catching banner with call-to-action button
3. **Services**: 6 core consulting services with icons
   - Strategy Consulting
   - Business Analytics
   - Process Optimization
   - Digital Transformation
   - Organization Development
   - Change Management

4. **About Section**: Company information with impressive statistics
5. **Team Section**: Expert team members showcase
6. **Contact Section**: Contact information and inquiry form
7. **Footer**: Links, social media, and copyright

### 🚀 Functionality

- Smooth scroll navigation
- Mobile hamburger menu
- Contact form with validation
- Scroll animations for elements
- Active navigation highlighting
- Social media links

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling with responsive design
- `script.js` - Interactive features and animations

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your company information:
   - Update company name, address, phone, and email
   - Modify team member names and roles
   - Update service descriptions
   - Change contact information

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1e3a8a;      /* Navy Blue */
    --secondary-color: #3b82f6;    /* Sky Blue */
    --accent-color: #f59e0b;       /* Gold */
    --text-color: #1f2937;
    --light-bg: #f9fafb;
}
```

### Content
Simply edit the text in `index.html`:
- Update company name ("BSS Group")
- Change service descriptions
- Modify team member information
- Update contact details
